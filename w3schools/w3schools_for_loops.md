[W3Schools exercises: For Loops](https://www.w3schools.com/python/python_for_loops.asp)
### Exercise 1
```python
fruits = ["apple", "banana", "cherry"]
for x in fruits:
	print(x)
```
### Exercise 2
```python
fruits = ["apple", "banana", "cherry"]
for x in fruits:
	if x == "banana":
		continue
	print(x)
```

### Exercise 3
```python
for x in range(6):
	print(x)
```

### Exercise 4
```python
fruits = ["apple", "banana", "cherry"]
for x in fruits:
	if x == "banana":
		break
	print(x)
```
